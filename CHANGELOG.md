# 更新日志

## [0.2.0] - 2024-12-02

### 新增功能 🎉
- ✨ **编辑功能**: 支持在查看器中直接编辑文本节点和 MD 文件
- ✨ **双模式编辑**: 支持源码模式和预览模式切换
- ✨ **工具栏**: 为文本节点和文件节点添加编辑工具栏
- ✨ **保存功能**: 文本节点保存到 Canvas,MD 文件保存到原文件
- ✨ **打开文件**: MD 文件节点可以快速在 Obsidian 中打开
- ✨ **设置选项**: 新增"启用编辑功能"和"默认编辑模式"设置

### 优化改进 📈
- 🎨 重新设计节点头部,添加工具栏布局
- 🎨 优化编辑器样式,支持自动调整高度
- 🎨 改进按钮样式和交互效果
- 💾 添加保存确认提示
- 🔄 编辑状态切换更流畅

### Bug 修复 🐛
- 🐛 修复文件节点 `filePath.startsWith` 错误
- 🐛 修复空值检查问题
- 🐛 改进文件路径解析逻辑

## [0.1.0] - 2024-12-02

### 新增功能
- ✨ 初始版本发布
- ✨ 支持监听 Canvas 节点选择
- ✨ 支持文本节点内容显示
- ✨ 支持 MD 文件节点内容显示
- ✨ 支持 Markdown 渲染
- ✨ 支持自定义查看器位置(左侧/右侧)
- ✨ 支持自动打开查看器功能
- ✨ 显示节点颜色标识
- ✨ 添加设置面板

### 技术实现
- 基于 Obsidian Canvas API
- 使用定时轮询检测节点选择状态
- 使用 ItemView 创建自定义侧边栏视图
- 使用 MarkdownRenderer 渲染内容

### 已知限制
- 当前仅支持文本节点和 MD 文件节点
- 轮询间隔固定为 500ms
- 不支持其他文件类型(如图片、PDF等)
- 不支持编辑功能
